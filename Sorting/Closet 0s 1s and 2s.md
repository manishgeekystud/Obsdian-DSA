Given an array of 0s, 1s, and 2s. Arrange the array elements such that all 0s come first, followed by all the 1s and then, all the 2s.

**Note:** Do not use the inbuilt sort function.  
 

**Example 1:**

**Input**: N = 5, arr[] = {0, 2, 1, 2, 0}
**Output**: 0 0 1 2 2

**Example 2:**

**Input**: N = 3, arr[] = {0, 1, 0}
**Output**: 0 0 1
