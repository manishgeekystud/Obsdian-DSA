
The **Left View** of a binary tree contains the **first node visible from the left side** at every level.

> **Interview Frequency:** ⭐⭐⭐⭐⭐ (Very Frequently Asked)

---

# Example

```
            10
          /    \
        20      30
       /  \    /  \
     40   50  60   70
          /
         80
```

### Left View

```
10 20 40 80
```

Why?

- Level 0 → **10**
- Level 1 → **20**
- Level 2 → **40**
- Level 3 → **80**

These are the first nodes seen from the left.

---

# Approach 1 (Recommended): Level Order Traversal (Queue)

### Idea

At every level:

1. Find the number of nodes (`size = queue.size()`).
2. Process all nodes in that level.
3. Print **only the first node** (`i == 0`).

---

# Algorithm

```
Insert root into queue

While queue is not empty

    size = queue.size()

    for i = 0 to size-1

         node = queue.poll()

         if i == 0
              print node

         insert left child
         insert right child
```

---

# Java Code
```java
import java.util.LinkedList;
import java.util.Queue;

class Node {

    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
    }
}

public class LeftView {

    static void leftView(Node root) {

        // Empty tree
        if (root == null)
            return;

        Queue<Node> queue = new LinkedList<>();

        // Insert root
        queue.offer(root);

        while (!queue.isEmpty()) {

            // Number of nodes at current level
            int size = queue.size();

            for (int i = 0; i < size; i++) {

                Node current = queue.poll();

                // First node of this level
                if (i == 0)
                    System.out.print(current.data + " ");

                // Add left child
                if (current.left != null)
                    queue.offer(current.left);

                // Add right child
                if (current.right != null)
                    queue.offer(current.right);
            }
        }
    }

    public static void main(String[] args) {

        /*
                    10
                  /    \
                20      30
               /  \    /  \
             40   50  60   70
                  /
                 80
        */

        Node root = new Node(10);

        root.left = new Node(20);
        root.right = new Node(30);

        root.left.left = new Node(40);
        root.left.right = new Node(50);

        root.right.left = new Node(60);
        root.right.right = new Node(70);

        root.left.right.left = new Node(80);

        leftView(root);
    }
}
```

# Why `i == 0`?

At every level:

```
Level 2

40 50 60 70
↑
First node
```

The **first node** processed in the level-order traversal is the **leftmost node**, so:

```
if (i == 0)
    System.out.print(current.data);
```

prints the left view.

---

# Time Complexity

Every node is visited exactly once.

```
Time Complexity = O(n)
```

---

# Space Complexity

The queue stores nodes level by level.

```
Space Complexity = O(n)
```

# Recursive Approach (Also Frequently Asked)

The recursive solution keeps track of the **maximum level visited so far**.

### Idea

- Traverse **Left subtree first**, then **Right subtree**.
- If the current level is visited for the first time, print the node.

### Java Code

```java
class LeftViewRecursive {

    static int maxLevel = -1;

    static void leftView(Node root, int level) {

        if (root == null)
            return;

        if (level > maxLevel) {
            System.out.print(root.data + " ");
            maxLevel = level;
        }

        // Visit left subtree first
        leftView(root.left, level + 1);

        // Then visit right subtree
        leftView(root.right, level + 1);
    }
}
```

where `h` is the height of the tree.

# Common Interview Questions

### 1. Why does `i == 0` print the left view?

Because in BFS, nodes are processed **left to right** within each level. The first node removed from the queue at each level is the leftmost node.

---

### 2. How do we print the Right View?

Use the same BFS approach, but print the **last node** at each level:

```
if (i == size - 1)
    System.out.print(current.data);
```

---

### 3. Why does the recursive solution visit the left subtree first?

The first node reached at a new level becomes part of the left view. Visiting the left subtree before the right ensures the leftmost node is discovered first.

---

# Interview Tip

Remember these three patterns:

|Problem|Condition|
|---|---|
|**Level Order Line by Line**|Print every node|
|**Left View**|Print when `i == 0`|
|**Right View**|Print when `i == size - 1`|
