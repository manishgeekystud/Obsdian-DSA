
The **Left View** of a binary tree contains the **first node visible from the left side** at every level.

> **Interview Frequency:** ⭐⭐⭐⭐⭐ (Very Frequently Asked)

---

# Example

```
            10
          /    \
        20      30
       /  \    /  \
     40   50  60   70
          /
         80
```

### Left View

```
10 20 40 80
```

Why?

- Level 0 → **10**
- Level 1 → **20**
- Level 2 → **40**
- Level 3 → **80**

These are the first nodes seen from the left.

---

# Approach 1 (Recommended): Level Order Traversal (Queue)

### Idea

At every level:

1. Find the number of nodes (`size = queue.size()`).
2. Process all nodes in that level.
3. Print **only the first node** (`i == 0`).

---

# Algorithm

```
Insert root into queue

While queue is not empty

    size = queue.size()

    for i = 0 to size-1

         node = queue.poll()

         if i == 0
              print node

         insert left child
         insert right child
```

---

# Java Code
```java
import java.util.LinkedList;
import java.util.Queue;

class Node {

    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
    }
}

public class LeftView {

    static void leftView(Node root) {

        // Empty tree
        if (root == null)
            return;

        Queue<Node> queue = new LinkedList<>();

        // Insert root
        queue.offer(root);

        while (!queue.isEmpty()) {

            // Number of nodes at current level
            int size = queue.size();

            for (int i = 0; i < size; i++) {

                Node current = queue.poll();

                // First node of this level
                if (i == 0)
                    System.out.print(current.data + " ");

                // Add left child
                if (current.left != null)
                    queue.offer(current.left);

                // Add right child
                if (current.right != null)
                    queue.offer(current.right);
            }
        }
    }

    public static void main(String[] args) {

        /*
                    10
                  /    \
                20      30
               /  \    /  \
             40   50  60   70
                  /
                 80
        */

        Node root = new Node(10);

        root.left = new Node(20);
        root.right = new Node(30);

        root.left.left = new Node(40);
        root.left.right = new Node(50);

        root.right.left = new Node(60);
        root.right.right = new Node(70);

        root.left.right.left = new Node(80);

        leftView(root);
    }
}
```
