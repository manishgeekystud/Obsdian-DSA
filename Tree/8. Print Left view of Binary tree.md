
The **Left View** of a binary tree contains the **first node visible from the left side** at every level.

> **Interview Frequency:** ⭐⭐⭐⭐⭐ (Very Frequently Asked)

---

# Example

```
            10
          /    \
        20      30
       /  \    /  \
     40   50  60   70
          /
         80
```

### Left View

```
10 20 40 80
```

Why?

- Level 0 → **10**
- Level 1 → **20**
- Level 2 → **40**
- Level 3 → **80**

These are the first nodes seen from the left.

---

# Approach 1 (Recommended): Level Order Traversal (Queue)

### Idea

At every level:

1. Find the number of nodes (`size = queue.size()`).
2. Process all nodes in that level.
3. Print **only the first node** (`i == 0`).

---

# Algorithm

```
Insert root into queue

While queue is not empty

    size = queue.size()

    for i = 0 to size-1

         node = queue.poll()

         if i == 0
              print node

         insert left child
         insert right child
```
