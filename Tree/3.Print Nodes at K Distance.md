Problem Statement: Given a binary tree and an integer k, print all nodes that are exactly k levels below the root.

### Example

```
            10
          /    \
         20     30
        / \     / \
      40  50   60  70
     /
    80
```

### Input
```
k = 2
```
### Output
```
40 50 60 70
```

Because these nodes are exactly **2 edges away** from the root.

---

# Idea

Think recursively.
At every node:

- If `k == 0`
    - We've reached the required distance.
    - Print the current node.
- Otherwise
    - Go one level deeper into both subtrees.
    - Decrease `k` by 1.

Every recursive call moves one level down.

---

# Recursive Formula

```
printKDistance(node, k)

If node == null
      return

If k == 0
      print node

Else
      print left with k-1
      print right with k-1
```

```java
class Node {
    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
    }
}

public class PrintNodesAtKDistance {

    // Prints all nodes that are k distance from the root
    static void printKDistance(Node root, int k) {

        // Base Case 1:
        // Tree is empty, so nothing to print.
        if (root == null)
            return;

        // Base Case 2:
        // We have reached the required level.
        if (k == 0) {
            System.out.print(root.data + " ");
            return;
        }

        // Go to the left subtree with one less distance
        printKDistance(root.left, k - 1);

        // Go to the right subtree with one less distance
        printKDistance(root.right, k - 1);
    }

    public static void main(String[] args) {

        /*
                    10
                  /    \
                 20     30
                / \    / \
              40  50  60  70
             /
            80

        */

        Node root = new Node(10);

        root.left = new Node(20);
        root.right = new Node(30);

        root.left.left = new Node(40);
        root.left.right = new Node(50);

        root.right.left = new Node(60);
        root.right.right = new Node(70);

        root.left.left.left = new Node(80);

        int k = 2;

        System.out.print("Nodes at distance " + k + " : ");

        printKDistance(root, k);
    }
}
```
