Given a binary tree, the task is to find the height of the tree. Height of the tree is the number of edges in the tree from the root to the deepest node, Height of the empty tree is **0**.  
 

![Example Tree](https://media.geeksforgeeks.org/wp-content/cdn-uploads/2009/06/tree122.gif)

Recursively calculate height of **left** and **right** subtrees of a node and assign height to the node as **max of the heights of two children plus 1**

> _Consider the following graph:_
> 
> _![Example Tree](https://media.geeksforgeeks.org/wp-content/cdn-uploads/2009/06/tree122.gif)_
> 
> _maxDepth(‘1’) = max(maxDepth(‘2’), maxDepth(‘3’)) + 1 = 2 + 1_
> 
> _because recursively_   
> _maxDepth(‘2’) =  max (maxDepth(‘4’), maxDepth(‘5’)) + 1 = 1 + 1 and  (as height of both ‘4’ and ‘5’ are 1)_  
> _maxDepth(‘3’) = 1_

Follow the below steps to Implement the idea:

- Recursively do a Depth-first search.
- If the tree is empty then return -1
- Otherwise, do the following
    - Get the max depth of the left subtree recursively  i.e. call maxDepth( tree->left-subtree)
    - Get the max depth of the right subtree recursively  i.e. call maxDepth( tree->right-subtree)
    - Get the max of max depths of **left** and **right** subtrees and **add 1** to it for the current node.

```java
class Node {
    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
    }
}

public class HeightBinaryTree {

    // Returns height in terms of number of nodes
    static int height(Node root) {

        // Base Case:
        // If tree is empty, height is 0.
        if (root == null)
            return 0;

        // Recursively calculate height of left subtree
        int leftHeight = height(root.left);

        // Recursively calculate height of right subtree
        int rightHeight = height(root.right);

        // Height of current node =
        // 1 (current node) +
        // maximum of left and right subtree heights
        return 1 + Math.max(leftHeight, rightHeight);
    }

    public static void main(String[] args) {

        /*
                  1
                 / \
                2   3
               / \
              4   5
             /
            6

        Height = 4 nodes
        */

        Node root = new Node(1);

        root.left = new Node(2);
        root.right = new Node(3);

        root.left.left = new Node(4);
        root.left.right = new Node(5);

        root.left.left.left = new Node(6);

        System.out.println("Height = " + height(root));
    }
}
```
