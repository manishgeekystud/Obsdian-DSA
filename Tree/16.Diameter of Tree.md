# Diameter of Binary Tree 

This is one of the **Top 10 Binary Tree Interview Questions**. Interviewers often expect you to know **both the brute-force and optimized approaches**, and to explain why the optimized solution is better.

---

# Problem Statement

Given the **root** of a binary tree, your task is to find the **diameter** of the binary tree. The **diameter** of a binary tree is defined as the number of edges on the **longest path between any two nodes**. Note that this path may or may not pass through the root of the tree.

### Diameter

The **diameter** is the **longest path between any two nodes** in the tree.

The path:

- May pass through the root.
- May not pass through the root.

# Important Observation

For every node,

```
Diameter through current node

=

Left Height

+

Right Height

+

Current Node
```

If counting nodes

```
Left Height + Right Height + 1
```

If counting edges

```
Left Height + Right Height
```

# Approach 1 — Brute Force (O(N²))

## Idea

For every node

1. Find left height.
2. Find right height.
3. Compute diameter through current node.
4. Recursively compute diameter in left subtree.
5. Recursively compute diameter in right subtree.
6. Return the maximum.

---

## Algorithm

```
For every node

↓

Find Left Height

↓

Find Right Height

↓

Current Diameter

↓

Left Diameter

↓

Right Diameter

↓

Maximum of all three
```

---

## Code

```java
class Solution {

    int diameter(Node root) {

        if (root == null)
            return 0;

        int leftHeight = height(root.left);
        int rightHeight = height(root.right);

        int current = leftHeight + rightHeight + 1;

        int leftDiameter = diameter(root.left);
        int rightDiameter = diameter(root.right);

        return Math.max(current,
                Math.max(leftDiameter, rightDiameter));
    }

    int height(Node root) {

        if (root == null)
            return 0;

        return 1 + Math.max(height(root.left),
                            height(root.right));
    }
}
```

# Approach 2 — Optimized DFS (Single Traversal)

**Most Asked Interview Solution**

---

## Key Idea

Calculate

- Height
- Diameter

in one recursion.

---

Instead of

```
Height()

↓

Diameter()
```

Do

```
Height

+

Update Diameter
```

inside same recursion.

---

## Algorithm

For every node

```
Left Height

↓

Right Height

↓

Update Diameter

↓

Return Height
```

---

## Code

```java
class Solution {

    int diameter = 0;

    int diameter(Node root) {

        height(root);

        return diameter;
    }

    int height(Node root) {

        if (root == null)
            return 0;

        int left = height(root.left);

        int right = height(root.right);

        diameter = Math.max(diameter,
                            left + right + 1);

        return 1 + Math.max(left, right);
    }
}
```
