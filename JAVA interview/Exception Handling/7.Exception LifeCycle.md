**Exception Lifecycle** is the complete journey of an exception from the moment an error occurs until it is handled or the program terminates.

In simple words:

> **Exception Lifecycle = Creation → Throwing → Propagation → Handling → Program Continues (or JVM Terminates)**

This is one of the most commonly asked interview topics because it explains **how the JVM internally handles exceptions**.
